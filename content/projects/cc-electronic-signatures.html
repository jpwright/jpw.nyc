Electronic Signing for Confluence Cloud

August 2017

Documentation

Signing a page

While viewing a document, click the menu (typically the three-dot icon which also has links to Move, Copy, Delete, etc.) and choose "Add Signature". You should see a dialog appear. Fill out the dialog with your email, password, "meaning", and (optional) comment.

The add-on will verify that the email and password match the presently logged in Confluence user. If not, the email and password fields are highlighted in red to indicate a problem. Note that too many invalid attempts may temporarily disable the ability to sign documents. (This is a restriction of the Confluence API.)

Viewing signatures

While editing a page, add the Signatures macro (click the plus-sign icon and choose Signatures). The macro should display all previous signatures, with timestamps and links to the profile page of the user that signed the document.

At this time it is not possible to edit or delete signatures.

Data security statement

Overview

The Electronic Signature add-on enables 21 CFR 11 compliant document signing on Confluence Cloud. Primarily, the add-on ensures that all signatures are authenticated at the time of signature, and have timestamps and “meaning” associated with them. All transactions with Confluence Cloud are secured using Atlassian Connect Express / JWT.

Data storage

The add-on does not store any data whatsoever. It serves three endpoints: the “Add Signature” dialog template, the “Signatures” macro, and an “authentication” endpoint used to “double check” Atlassian username and password credentials.

Certification

The add-on is not certified by any authority. 21 CFR 11 compliance is a complex process, and this add-on is merely a small part. Full compliance will depend on Confluence settings, user management, and other operational processes.

Backups

Because no data is stored, no backups take place.

Privacy

All transactions are secured using HTTPS/JWT. Because no user data is stored, there is no long-term privacy risk to using this add-on.
